Игра основана на первой игре серии the legend of zelda. В игре представлены такие механики как ходьба и примитивная боевка. Игрок перемещается на стрелочках, а бъет на пробел. Чтобы начать драться нужно сперва взять меч в пещере. Чтобы пройти игру нужно убить босса. Графика в игре сделана при помощи примитивных фигур и заливки. У врагов в игре есть их собственное движение, обычные враги будут ходить по своему пути, а босс будет идти на игрока. Также для убийства обычных врагов хватит одного удара, а для убийства босса уже понадобится больше.
